#### Byte to String
```java
jshell> String s = "Hello World";
s ==> "Hello World"

jshell> byte[] byteArray = s.getBytes();
byteArray ==> byte[11] { 72, 101, 108, 108, 111, 32, 87, 111, 114, 108, 100 }

jshell> new String(byteArray)
$1 ==> "Hello World"
```
